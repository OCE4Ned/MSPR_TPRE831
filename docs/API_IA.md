# API IA

L'api IA est développée en Python et utilise FastAPI pour créer une interface de programmation d'applications (API) qui permet d'interagir avec les fonctionnalités d'intelligence artificielle du projet. Cette API est conçue pour être utilisée par le frontend de l'application, permettant ainsi une communication fluide entre les différentes parties du projet.

## Installation

```bash
python -m venv .venv (<- nom de votre environnement virtuel)
```

```bash
pip install -r requirements.txt
```

## Lancement de l'application

```bash
python app/main.py
```
