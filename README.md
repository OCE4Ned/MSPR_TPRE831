# MSPR_TPRE831
Le repo git pour le Bloc 3 (TPRE831) d'évaluation de notre MSPR pour la première année de master EISI IA

## Fonctionnement du repo
Le repo est organisé en plusieurs dossiers : `backend`, `frontend`, `api_ia` et `docs`.
- Le dossier `backend` contient le code source de l'application backend, qui est développée en Python avec le framework FastAPI. Il inclut également un fichier `requirements.txt` pour gérer les dépendances du projet.
- Le dossier `frontend` contient le code source de l'application frontend, qui est développée en React. Il inclut également un fichier `package.json` pour gérer les dépendances du projet.
- Le dossier `api_ia` contient le code source de l'API d'intelligence artificielle, qui est développée en Python. Il inclut également un fichier `requirements.txt` pour gérer les dépendances du projet.
- Le dossier `docs` contient la documentation du projet, avec des fichiers Markdown détaillant les étapes d'installation, de configuration et d'utilisation pour le backend, le frontend et l'utilisation de Docker.
